# 🕉️ Mahakumbh Mela 2K25 Analysis Dashboard

An interactive **Power BI Dashboard** project analyzing the **Mahakumbh Mela 2025** dataset using data analytics and visualization techniques.
This project explores visitor sentiments, media coverage, revenue generation, and location-based insights related to the world's largest spiritual gathering.

---

## 📌 Project Overview

The **Mahakumbh Mela 2025 Analysis Dashboard** provides a comprehensive analytical view of:

* 📈 Revenue generation trends over the years
* 😊 Positive and negative sentiment analysis
* 🌍 International media coverage
* 📍 Location-wise revenue distribution
* 📊 Interactive KPI monitoring

The dashboard was designed to transform raw festival-related data into meaningful insights using **Power BI**.

## 📊 Key Insights

### 💰 Revenue Analysis

* Prayagraj generated the highest overall revenue among analyzed locations.
* Revenue trends indicate significant growth during major Mahakumbh years.

### 😊 Sentiment Analysis

* Positive sentiment rate reached approximately **80%**.
* Negative sentiment fluctuations were observed during peak crowd years.

### 🌍 Media Coverage

* International media attention increased significantly during recent Mahakumbh events.

### 📍 Location-Based Insights

Locations analyzed:

* Prayagraj
* Haridwar
* Nashik
* Ujjain
